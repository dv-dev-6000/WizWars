========
Wiz Wars
========

* Launch WizWars.exe to play

* Requires a controller

* Instrustions for play available in game

* press back(select) to quit


This project was created with monogame in c# as part of an object oriented programming module undertaken at Edinburgh College.
The aim was to create a bomberman clone themed around wizards  duelling in a castle.

The project is the sole work of Daniel Vermeulen and is published on github for portfolio purposes.
